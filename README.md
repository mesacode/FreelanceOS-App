# FreelanceOS

Freelancerlar için geliştirilmiş modern masaüstü CRM ve iş yönetim platformu.
Müşterilerinizi, finanslarınızı, hatırlatıcılarınızı ve iş süreçlerinizi tek panelden yönetin.

---

## Özellikler

### CRM

* Müşteri yönetimi
* Müşteri detay sayfaları
* Arama / filtreleme / sıralama
* Duplicate kayıt engelleme
* Durum takibi

### Finans Yönetimi

* Gelir / Gider takibi
* Aylık özetler
* Net bakiye hesaplama
* Müşteri bağlantılı finans kayıtları

### Hatırlatıcı Sistemi

* Görev / Hatırlatma oluşturma
* Tamamlandı takibi
* Tarih filtreleme
* Müşteri bağlantılı hatırlatmalar

### WhatsApp Şablonları

* Şablon yönetimi (CRUD)
* Hızlı erişim mesaj sistemi
* Duplicate şablon koruması

### Dashboard

* Aylık gelir / gider görünümü
* Net bakiye
* Aktif müşteri sayısı
* Bekleyen hatırlatmalar
* Son aktiviteler

### Ayarlar

* Dinamik uygulama ayarları
* Kayıt / Güncelle / Sil / Ara desteği
